Questions

Is BYOM really free?

Yes. You install the extension and MCP server locally, bring your own AI model API key, and run everything on your machine. Hanzi doesn't charge for BYOM use.

Is the API really free right now?

Yes, during early access. You can sign in, create an API key, pair a browser, and run tasks at no cost. We track usage internally but do not charge during this period. Usage-based pricing will be introduced later with advance notice.

How do I get managed access?

Managed access is available by request. We handle model routing so you don't need your own API key. Contact us to get started. Billing for managed access is not yet active.

What will pricing look like after early access?

We expect usage-based pricing for the API (per task) and a monthly subscription for managed access. Exact pricing is not finalized. Early access users will get advance notice before any billing starts.

Can I switch between modes?

Yes. BYOM and managed/API access are separate. You can use BYOM for personal tasks and the API for your product.
