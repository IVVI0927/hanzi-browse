## ✨ What can it do?

> Your AI agent can:
> - Click buttons
> - Fill forms
> - Access logged-in pages
> - Complete multi-step workflows

## What is this

Hanzi Browse bridges the gap between AI agents and real web browsing. Instead of limited text-based web access, your AI agent gets a full Chrome browser with your existing logins and sessions.

**Two ways to use it:**
- **For AI users**: Install locally, your agent controls your browser directly
- **For developers**: API integration, embed browser automation in your product

## Why it matters

Most AI agents are blind to the modern web — they can't handle authentication, JavaScript, or complex user interfaces. Hanzi Browse changes that by giving agents access to a real browser environment.
